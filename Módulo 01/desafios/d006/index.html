Tags em HTML

Em HTML5, podemos inserir os elementos das lista a seguir. Passe o mouse sobre os termos para descobrir qual tag deverá utilizar em cada uso.

  • Parágrafo
  • Títulos
  • Imagens
  • Código-fonte (Informa que o código fonte da página está escrito em HTML5)
  • Lista Ordenada/Numerada (Pode ser em números; números romanos; ordem alfabética maiscúla e minuscúla)
  • Lista não ordenada/Demarcação (Os marcadores podem ser circulo fechado; circulo aberto e quadrado)
